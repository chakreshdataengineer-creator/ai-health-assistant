# AI Health Assistant

AI-powered health and fitness assistant built using FastAPI and OpenAI.

## Features

- AI meal analysis
- Daily calorie tracking
- Water intake tracking
- BMI calculator
- Personalized meal plans
- Autonomous water reminders
- Health warnings
- SQLite database memory

## Tech Stack

- Python
- FastAPI
- OpenAI API
- SQLite
- APScheduler
